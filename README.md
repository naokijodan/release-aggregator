# リリース速報Pro — セットアップガイド

## リリース速報Proとは

さまざまなブランドやメーカーの新商品・リリース情報を自動で集めて、見やすく一覧表示するツールです。
気になるジャンルやキーワードで絞り込んで、最新情報をすばやくチェックできます。

---

## 必要なもの

| 必要なもの | 備考 |
|-----------|------|
| パソコン（Windows / Mac） | どちらでも動作します |
| Docker Desktop（無料） | アプリを動かすための土台となるソフトです |

---

## インストール手順（Mac向け）

### Step 1: Docker Desktop をインストールする

1. 以下のページをブラウザで開きます

   https://www.docker.com/products/docker-desktop/

2. **「Download for Mac」** ボタンを押して、インストーラーをダウンロードします

3. ダウンロードされたファイル（`Docker.dmg`）をダブルクリックして開きます

4. 表示されたウィンドウで、Docker のアイコンを「Applications」フォルダにドラッグ＆ドロップします

5. 「アプリケーション」フォルダから **Docker** を起動します

6. 初回は「アクセスを許可しますか？」などの確認が出るので、すべて **許可** を選んでください

7. 画面の上部（メニューバー）にクジラのアイコンが表示され、「Docker Desktop is running」と出れば準備完了です。起動まで1〜2分ほどかかることがあります

---

### Step 2: リリース速報Pro をダウンロードする

1. 以下のページをブラウザで開きます

   `https://github.com/xxxxx/release-aggregator`
   （※ URLは後日お知らせします）

2. 緑色の **「Code」** ボタンを押し、**「Download ZIP」** を選びます

3. ダウンロードされた ZIP ファイルをダブルクリックして展開します

4. 展開されたフォルダ（`release-aggregator-main` のような名前）を、分かりやすい場所に移動します
   - おすすめ: デスクトップ、または「書類」フォルダ

---

### Step 3: 起動する

1. **ターミナル**（コンピュータに文字で指示を出すためのアプリ）を開きます
   - 「アプリケーション」→「ユーティリティ」→「ターミナル」の順にクリック
   - または、画面右上の虫めがねアイコン（Spotlight）で「ターミナル」と検索しても開けます

2. ターミナルの画面に、以下のように入力して Enter キーを押します

   ```
   cd ~/Desktop/release-aggregator-main
   ```

   ※ フォルダを「書類」に置いた場合は `cd ~/Documents/release-aggregator-main` と入力してください。
   フォルダ名が異なる場合は、実際のフォルダ名に読み替えてください。

3. 続けて、以下を入力して Enter キーを押します

   ```
   ./start.sh
   ```

4. 初回は必要なデータのダウンロードが行われるため、**数分ほど時間がかかります**。
   画面に文字がたくさん流れますが、正常な動作です。そのままお待ちください。

5. 「started」や「ready」といったメッセージが表示されたら起動完了です

---

### Step 4: ブラウザで開く

1. お使いのブラウザ（Safari、Chrome など）を開きます

2. アドレスバーに以下を入力して Enter キーを押します

   ```
   http://localhost:3000
   ```

3. リリース速報Pro の画面が表示されれば成功です

---

## 使い方

### カテゴリの切り替え

画面上部のタブをクリックすると、ジャンルごとにリリース情報を切り替えて見ることができます。

### キーワード検索

検索ボックスにキーワードを入力すると、そのキーワードを含む情報だけに絞り込めます。

### 注目ワード・ブランドでの絞り込み

画面に表示されている注目ワードやブランド名をクリックするだけで、関連する情報に絞り込めます。

### 管理画面で情報源やカテゴリを追加する

1. ブラウザで以下のアドレスを開きます

   ```
   http://localhost:3000/admin
   ```

2. 管理画面から、新しい情報源（ニュースサイトなど）やカテゴリを追加できます

---

## 停止方法

使い終わったら、以下のどちらかの方法で停止できます。

### 方法 1: ターミナルから停止する

ターミナルを開いて、起動時と同じフォルダで以下を入力します。

```
./stop.sh
```

### 方法 2: Docker Desktop から停止する

1. Docker Desktop アプリを開きます
2. 「Containers」の一覧に表示されているリリース速報Pro の項目を見つけます
3. 停止ボタン（四角いアイコン）をクリックします

## よくある質問

### 「起動できない」「エラーが出る」

- **Docker Desktop が起動しているか確認してください。** 画面上部のメニューバーにクジラのアイコンが表示されていれば起動しています。表示されていなければ、「アプリケーション」から Docker を起動してください。
- Docker Desktop を起動した直後は準備に時間がかかります。クジラのアイコンが動いている間は、しばらくお待ちください。

### 「画面は開けるが、情報が表示されない」

- 初回起動時は、各サイトから情報を集めるのに少し時間がかかります。数分待ってからページを再読み込み（リロード）してみてください。

### 「新しいジャンルやカテゴリを追加したい」

- 管理画面（`http://localhost:3000/admin`）から追加できます。画面の案内に従って、カテゴリ名や情報源のURLを入力してください。

### 「パソコンを再起動したら動かなくなった」

- パソコンを再起動すると Docker Desktop も停止します。Docker Desktop を起動してから、Step 3 の手順でもう一度 `./start.sh` を実行してください。
